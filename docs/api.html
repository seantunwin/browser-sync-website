Our API is incredibly simple & powerful. You can use it to create your own tiny node program for local development or integrate with other tools.

1. Initialize

To use our API in any NodeJS program, simply require the BrowserSync module like you would any other. This will give you access to the public methods detailed below.

{% highlight javascript %} var browserSync = require("browser-sync"); {% endhighlight %}

.init([options, callback])

Calling this method will always start a BrowserSync server and if you provide no options, BrowserSync will start in snippet mode and will require you to copy/paste the provided code into your website. Don't forget you can review all available options here.

{% highlight javascript %} {% include scripts/api/init.js %} {% endhighlight %}

The main exported function is also an alias to .init() - so if you prefer, you can also do this:

{% highlight javascript %} var browserSync = require("browser-sync"); browserSync(); {% endhighlight %}

You'll more than likely want to run a server or proxy when using BrowserSync - you specify which by passing an options object as the first parameter:
